This page is dedicated to showcasing in-depth phantom job guides written by various authors through out the N.A. community. 
These are both for learning, easing more people into the role, and further optimization of those said roles.

Some phantom jobs have macros below with placeholders already set up. If needed, you can modify the placeholder (eg. `<bind1>` and `<f>`) with the following depending on your needs.
If you want a full list of placeholders please visit the <a href="https://na.finalfantasyxiv.com/lodestone/playguide/db/text_command/placeholder/" target="_blank">Eorzea Database page for them.</a>
- `<f>` for a focused target
- `<bind1>` for a character marked with the Bind 1 head marker
- `<attack1>` for a character marked with the Attack 1 head marker
- `<stop1>` for a character marked with the Ignore 1 head marker
- `<cross>` for a character marked with the Cross/Plus head marker
- `<1-8>` for someone in your party.
- `<tt>` for the target of your current target.

## Geomancer
#### Battle Bell and You: How to Help Your Geomancer Feel Useful
By Mortia Starfallgirl. Contains timings on when to take damage or strategic thrice come ruins to maintain Battle Bell's damage buff.
- <a href="https://docs.google.com/spreadsheets/d/1k2eIlDWksPi_fTXIRRlCzeAm2m9GewHqFpnShlOgg1Q/edit?usp=sharing" target="_blank">LINK TO GUIDE</a>

### Macros
Helpful macro templates to maintain bells on two people. It will remind you when to use the next Battle Bell so you
can maintain 100% uptime. Useful to put on for Main Tanks and Berserkers.

**First Bell Target**
```
/gaction "Phantom Action I" <bind1>
/gaction "Phantom Action I" <bind1>
/gaction "Phantom Action I" <bind1>
/gaction "Phantom Action I" <bind1>
/e Bell given to <bind1> (macro 1). <wait.30>
/e Use Bell Macro 2 next. <se.2>
/micon "Phantom Action I" general
```

**Second Bell Target**
```
/gaction "Phantom Action I" <bind2>
/gaction "Phantom Action I" <bind2>
/gaction "Phantom Action I" <bind2>
/gaction "Phantom Action I" <bind2>
/e Bell given to <bind2>(macro 2). <wait.30>
/e Use Bell Macro 1 next. <se.2>
/micon "Phantom Action I" general
```
